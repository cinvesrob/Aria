Detailed Description

template<class P1, class P2, class P3, class P4, class P5>
class ArGlobalFunctor5< P1, P2, P3, P4, P5 >

Functor for a global function with 5 parameters.

This is a class for global functions which take 5 parameters. This ties a C style function pointer into the functor class hierarchy as a convience. Code that has a reference to this class and treat it as an ArFunctor can use it like any other functor.

For an overall description of functors, see ArFunctor.
