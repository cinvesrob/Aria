void ArLineFinder::setMaxDistBetweenPoints ( int  maxDistBetweenPoints = 0) [inline]

Sets a maximum distance for points to be included in the same line.

If points are greater than this distance apart, then they will not be considered in the same line. If this value is 0, then there is no maximum-distance condition and points will be considered part of a line no matter how far apart.
