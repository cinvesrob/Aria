Detailed Description

template<class T>
class ArConstFunctorC< T >

Swig doesn't like the const functors.

Functor for a const member function This is a class for const member functions. This class contains the knowledge on how to call a const member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.

Java and Python Wrappers: Not available in Java or Python wrapper libraries.
