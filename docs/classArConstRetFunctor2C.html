Detailed Description

template<class Ret, class T, class P1, class P2>
class ArConstRetFunctor2C< Ret, T, P1, P2 >

Functor for a const member function with return value and 2 parameters.

This is a class for const member functions which take 2 parameters and return a value. This class contains the knowledge on how to call a member function on a particular instance of a class. This class should be instantiated by code that wishes to pass off a functor to another piece of code.

For an overall description of functors, see ArFunctor.
